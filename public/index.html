About Me

My name is Jay and I love to build things on the web. In 2003 I attended Branford Hall Career Institute for Computer Networking Management. This is where I was first introduced to very basic web development. I learned HTML and CSS, just enough to build static websites, but I really enjoyed it. Building websites was a hobby of mine and I would build websites for my friends bands and businesses. Recently I decided to change careers and wanted to learn how to build modern and complex web applications professionally. I recently completed the Full Stack Web Development program offered through The University of Connecticut School of Engineering. I have listed some of the skills I have aquired below.

Javascript

JavaScript is my strongest programming language and I have experience with MERN stack using javascript for both front end and backend applications. I have built many small applications and worked on several mid-sized projects using vanilla javascript, jquery and the React front end framework.

React

My experience with React includes building full stack applications using both class-based and functional components. I am proficient with JSX, React Router and implementing custom css and javascript into React applications. I have also used context api and hooks in the development of React apps.

Node

I have built many apps using node and express for backend development. I am adept at configuring api/html routes with express and installing npm packages within the node environment. Additionally I have experience with git version control including deployment of full stack applications to github and heroku. I built the backend for this website using node and express and it's hosted on heroku.

HTML5 & CSS

I have a great deal of experience with HTML markup and I have used several css frameworks. I am very familiar with Bootstrap and have built several applications using this framework. I have also used semantic UI and most recently have been learning Tailwind. In addition to these frameworks, I also enjoy using CSS grid. In fact, the front end for this website was built from stratch using HTML, CSS grid and vanilla javascript.

Wordpress

Wordpress is a great way to quickly build a CMS website that can easily be scaled up. I have experience creating wordpress sites and with my knowledge of web development I am able to customize themes to accommodate individual needs. I know some basic php which is helpful for customizing sites.

MongoDB and mySQL

MongoDB and mySql are the databases that I am familiar with. I have built several applications using mongoDB and the Mongoose ODM library. I have also built applications that use mySql including Wordpress CMS websites. I have created one full stack application using the Sequelize ORM.

Portfolio

Age Safe at Home


view github repository

This is a MERN full stack website that I built from scratch. This website is built for a fictional company called Age Safe at Home. This site matches user input with data stored in a mongoDB database. I used a css boilerplate called skeleton css for the basic grid layout and all other styling was hand coded. The website also uses twilio autopilot for the chatbot. I utilzed the zip-codes api and vanilla javascript to determine a radius to match appropriate providers with customers.

Weather Dashboard


view github repository

This project is a weather dashboard that uses the openweathermap API. You can enter any city name and it will return today's weather plus the 5 day forecast. Local storage is used to store previous search results and when the page is refreshed or re-opened, it will display the weather and 5 day forecast for based on your last search. All of the functionality is thanks to JavaScript. I used the openweather API to retrieve 5 day weather information and wrote the code to pull out the information I wanted and plug it in to the html file as necessary. In this project I created a very basic html file and then used JavaScript to dynamically create the content.

Budget Tracker


view github repository

This is a full stack progressive web application using node, express, mongodb, mongoose, and indexedDB. This web application tracks deposits and withdrawals and can be downloaded as a local application and will maintain its functionality offline. My role for this project was implementing mongoDB and indexedDB, as well as configuring the web manifest to make this a fully operational PWA.

Team Profile Generator


view github repository

This is a command line application that generates an html page based on user input. The application runs using Node.js and Inquirer.js. The app is designed to allow the user to add as many team members as they want. When the user is finished entering information, a team.html page will be generated in the output directory and will display each team member as either a manager, engineer or intern. The project utilizes both Bootstrap and custom css for styling.
