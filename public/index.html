Portfolio

Age Safe at Home


view GitHub repository

This is a MERN full stack website that I built from scratch for a fictional company called Age Safe at Home. The site was created using React for the front-end, Node.js for server, and MondoDB for the database. The site uses Skeleton CSS for the basic grid design and custom CSS to style all other elements. Twilio Autopilot was used for the chatbot. I utilized the Zip-Codes API and vanilla JavaScript to determine a radius to match the user's location with the provider's location.

Weather Dashboard


view GitHub repository

This project is a weather dashboard that uses the openweathermap API. You can enter any city name and it will return today's weather plus the 5 day forecast. Local storage is used to store previous search results and when the page is refreshed or re-opened, it will display the weather and 5 day forecast for based on your last search. All of the functionality is thanks to JavaScript. I used the openweather API to retrieve 5 day weather information and wrote the code to pull out the information I wanted and plug it in to the html file as necessary. In this project I created a very basic html file and then used JavaScript to dynamically create the content.

Team Profile Generator


view GitHub repository

This is a command line application that generates an html page based on user input. The application runs using Node.js and Inquirer.js. The app is designed to allow the user to add as many team members as they want.
